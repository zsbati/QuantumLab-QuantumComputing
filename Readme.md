# QuantumLab - Quantum Computing Simulator
Interactive web-based quantum computing simulator.
## Features
- Drag-and-drop circuit builder
- Real-time quantum simulation  
- State visualization
- Pre-built algorithms
- Performance benchmarking
## Quick Start
1. Open [index.html](cci:7://file:///c:/Users/35196/beads_first/physics_sandbox/quantum_computing_simulator/index.html:0:0-0:0)
2. Drag gates to build circuit
3. Click "Run Circuit"
4. View results
## Gates
Single: H, X, Y, Z, S, T, Rx, Ry, Rz
Multi: CNOT, CZ, SWAP
## Shortcuts
Ctrl+R: Run, Ctrl+Delete: Clear, Ctrl+S: Export
## Files
- index.html - Main interface
- styles.css - Styling
- quantum-core.js - Core math engine
- quantum-gates.js - Gate definitions
- quantum-simulator.js - Circuit simulator
- visualization.js - State visualization
- app.js - Main application
